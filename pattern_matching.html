Pattern matching

Patterns are described with regular expressions. RegExp() is the constructor for creating objects that represent textual patterns.

  1. /^HTML/ : Matches the leters HTML at the start of a string
  2. /[1-9][[0-9]*/ : Matches a non-zero digit, followed by any # of digits
  3. /\bjavascript\b/i : Match "javscript" as a word, case-insentive
  • var sample_1 = "testing: 1, 2 ,3"; //our sample text
  • var pattern = /\d+/g //matches all instances of one or more digits
  • pattern.test(sample_1) //true: a match exists
  • sample_1.search(pattern) //9: index where condition first validates to true
  • sample_1.match(pattern) //=>["1", "2", "3"]; array of all matches
  • sample_1.replace(pattern, "#"); //=>: "testing: #, #, #"
  • sample_1.split(/\D+/); //=> ["", "1", "2", "3"]: split on non-digits
