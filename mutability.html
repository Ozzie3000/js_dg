mutable object references and immutable primitive values

Objects an arrays are changeable. undefined, null, booleans, numbers, and strings are not able to be changed or mutate.

    example 1
  1. var obj = {x=1}; //create an object
  2. obj.x = 2; //Mutate object; explicitly change value of property for x
  3. obj.y = 3; //Mutate object again. add new property y.
    example 2
  1. var array_1 = [1,2,3]; //Array mutable
  2. array_1[0] = "apple"; //change teh value of an array element. index 0
  3. array_1[3] = "car"; //add new array element. index 3.
    example 3
  1. var obj_1 = {x:1}, obj_2 = {x:1};
  2. or
  3. var obj_1 = {x:1}; var obj_2 = {x:1};
