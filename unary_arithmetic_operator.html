unary arithmetic operators

  • Unary plus (+)
  • Unary minus (-)
  • Increment (++)
  • Decrement (--)
    Increment(++) and Plus (+)
  • var a = 1, b = ++a;   a is now 2 and b is now 2
  • var c = 1, d = c++;   c is now 2 and d is 1
  • +"7";   converts operand to a number. 7
  • +7;   converts operand to a number, but in this case it already is. 7
    Decrement (--) and Minus (-)
  • var e = 1, f = --e;   e is now 0 and f is now 0
  • var g = 1, h = e--;   g is now 0 and h is 1
