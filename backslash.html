the \ character

Sequence Character represented
\o The Nul character
\b Backspace
\t Horizontal tab
\n Newline
\v Vertical tab
\f Form feed
\r Carriage return
\" Used to escape double quote
\' Used to escape single quote or apostrophe
\\ Used to escape backslash
\xXX The Latin-1 charater specified by two hexadecimal digits XX
\uXXX The Unicode characdter specified by the four hexadecimal digits XXXX
\ with any other character the backslash will be ignored
